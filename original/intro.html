Project Description

This system is a real-time events monitoring and data analysis system, which is based on network alerts data generated by Snort. The intention of this system is to support network administrator with handy, user-friendly web interface for accessing and analyzing Snort data.

Snort Introduction

Snort is an open source network intrusion prevention and detection system (IDS/IPS) developed by Sourcefire. Combining the bebefits of signature, protocol, and anomaly-based inspection. Snort is the most widely deployed IDS/IPS technology worldwide, and it has become the de facto standard for IPS.

Data Source

The snort security alerts data used in this project is taken from SnIPS project. This project sets up a mirror port on the K-state CIS department network, records all the traffic from May 2012 to Jan and farwards them to snort analysis interface. The snort analysed data are stored in a mysql database.
